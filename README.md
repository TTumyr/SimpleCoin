# SimpleCoin 0.1.1

Money tracking addon for World of Warcraft retail (9.0.2) and classic (1.13.5)

## Installation

The ZIP file contains 2 ZIP archives:

"\_retail\_.ZIP"

"\_classic\_.ZIP"

Unpack the relevant ZIP file in the respective WoW addon folder
